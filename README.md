# NASA Project

## Description
This project simplifies working with NASA's public datasets on space exploration, celestial phenomena, and more. It provides tools to analyze data, visualize results, and contribute to scientific discoveries.

## Features
- Real-time API integration with NASA's data sources.
- Easy-to-use command-line interface for dataset exploration.
- Data visualization tools for maps, star charts, and graphs.
